To run C++ program in terminal:
-------------------------------
cd Desktop/
touch filename.cpp
g++ -o filename filename.cpp -fopenmp
./filename



To run a c openmp program in terminal:
--------------------------------------
cd desktop/
touch filename.c
gcc -fopenmp filename.c -o filename
./filename
